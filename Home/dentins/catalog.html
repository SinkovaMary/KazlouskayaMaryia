Набор из 5 инструментов моделировочных для воска Томас Silicon

Доступность: Есть в наличии

11 060 ₽

10 760 ₽

1
  • Описание
  • Характеристики

Базовый набор для зубных для зубных техников, отлично подходит для воскового моделирования зубов.

В набор входит:

  1. Инструмент моделировочный для воска Томас (Thomas) №1 серый Silicon
  2. Инструмент моделировочный для воска Томас (Thomas) №2 красный Silicon
  3. Инструмент моделировочный для воска Томас (Thomas) №3 голубой Silicon
  4. Инструмент моделировочный для воска Томас (Thomas) №4 зеленый Silicon
  5. Инструмент моделировочный для воска Томас (Thomas) №5 оранжевый Silicon

Инструменты изготовлены из нержавеющей стали марки AISI 410, которая очень пластична, при этом стойко переносит воздействие высоких температур и не подвергается коррозии даже в агрессивной среде.

Производитель: Ul Amin, Пакистан.
Материал: нержавеющая сталь марки AISI 410 и силикон.
